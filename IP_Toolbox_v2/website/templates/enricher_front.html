Welcome to the IP Address enrichment tool

To use this tool you need to provide a list of IP addresses in a csv file, and specify a name for the output which also needs to be an excel document.

The file needs to have a heading of 'IP' and a list of IP addresses in the rows beneath this heading, each IP address in its own cell. Additional columns are optional and the result columns will be appended next to these.

The tool can potentially take some time to run, please expect to wait around 5 minutes for every 100 IP addresses submitted. When your request has been processed you will be redirected to a page where you download your results.

Remember both input file and output file name need to be in .csv format
